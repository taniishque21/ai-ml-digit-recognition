# ai-ml-digit-recognition

This project uses a Neural Network to recognize handwritten digits (0–9) using the MNIST dataset.

## Features
- Trains a neural network using TensorFlow/Keras
- Achieves ~97% accuracy
- Predicts handwritten digits from images
- Displays results using matplotlib
- Saves trained model for future use

---

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

---

##  Dataset
- MNIST dataset (built into TensorFlow)
- 60,000 training images
- 10,000 testing images
- Image size: 28x28 pixels
